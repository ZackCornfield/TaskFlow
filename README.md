# TaskFlow

TaskFlow is a task management application built with Angular for the frontend and ASP.NET Core for the backend. It allows users to manage boards, tasks, and team members efficiently. The application is designed to be scalable, secure, and user-friendly.

---

## Features

### Frontend

- Built with **Angular**.
- Responsive design for seamless use across devices.
- Manage boards, tasks, and team members.
- Integration with a backend API for dynamic data.

### Backend

- Built with **ASP.NET Core**.
- **Entity Framework Core** for database management.
- **JWT Authentication** for secure user access.
- RESTful API design.

---

## Project Structure

### Frontend

Located in the `frontend/` directory:

- **Angular CLI** project structure.
- Environment-specific configurations in `src/environments/`.
- Services for API communication.

### Backend

Located in the `TaskFlowApi/` directory:

- **ASP.NET Core** Web API project.
- `appsettings.json` for configuration.
- `Controllers/` for API endpoints.
- `Services/` for business logic.
- `Data/` for database context and migrations.

---

## Prerequisites

### Frontend

- Node.js (v16 or higher)
- Angular CLI

### Backend

- .NET SDK (v9.0 or higher)
- PostgreSQL database
